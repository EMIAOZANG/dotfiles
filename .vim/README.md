## Installation
    
	mv ~/.vim ~/.vim.orig 
	mv ~/.vimrc ~/.vimrc.orig
    git clone git://github.com/mckelvin/dotfiles.git ~/.dotfiles
    ln -sf ~/.dotfiles/.vim ~/.vim
	ln -sf ~/.vim/.vimrc ~/.vimrc

more: [gmarik/vundle/README.md](https://github.com/gmarik/vundle/blob/master/README.md)
